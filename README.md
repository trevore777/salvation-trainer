# Salvation Trainer — Full Upgrade

A Node.js + Express + EJS web app for teaching a KJV-only salvation course.

## Features

- Foundation module: Why the Bible is the authority
- Six lesson pathway
- Lesson unlock flow using browser localStorage
- Slide-style lesson pages
- Video lesson placeholders
- Workbook prompts
- Flashcard mode
- Challenge training mode
- Student progress tracker
- Teacher dashboard summary view
- KJV-only scripture references

## Run locally

```bash
npm install
npm run dev
```

Open:

```bash
http://localhost:3000
```

## Deploy

Works on Render, Railway, or Vercel Node server deployment.

For Vercel, use a simple Node server deployment or adapt to serverless routes if needed.
